IntelliJ 단축키 15가지

출처 : https://www.youtube.com/watch?v=QYO5_riePOQ
(windows에서의 단축키를 다루고 있습니다)

1. Alt + Enter

코드에서 오류가 발견된 경우 해당 오류 위에 커서를 놓고 Alt+Enter를 누르면 문제에 대한 수정 제안 목록이 표시됩니다.

2. F2

F2를 활용해 다음 오류, 경고, 제안으로 이동할 수 있습니다.

3. Alt + 1

Alt+1을 사용해 프로젝트 창을 열 수 있습니다.

4.ESC

ESC를 누르면 에디터로 돌아와 다시 코드에 집중할 수 있습니다.

5. Ctrl + E

Ctrl+E를 사용에 가장 최근에 열었던 파일을 확인할 수 있습니다.

6. Ctrl + B

Ctrl+B를 사용해 심볼의 선언으로 이동할 수 있습니다.

7. Alt + F7

선언을 검색하는 대신 사용 위치를 검색하고 싶은 경우, Alt+F7을 누르면 모든 사용 위치가 표시됩니다.

8. Tab Ctrl Twice

IDE에서 어떤 위치에 있든 혹은 어떤 파일이 열려 있든, Ctrl을 두 번 누르면 Run Anything 창이 열립니다.

9. Ctrl + W / Shift + Ctrl + W

커서 주변의 코드 섹션을 확장하거나 축소할 수 있습니다.

10. Ctrl + /

줄 어디서든 Ctrl+/를 누르면 코드가 줄 주석으로 처리됩니다.

11. Shift+Ctrl+Enter

현재 구문 완성 기능입니다.

12. Ctrl + Alt + L

간편하게 현재 파일의 서식을 프로젝트 표준에 맞출 수 있습니다.

13. Shift + Ctrl + Alt + T

모든 리팩토링을 표시할 수 있습니다.

14. Shift + Ctrl + A

IntelliJ IDEA의 모든 액션을 검색할 수 있습니다.

15. Tab Shift Twice

Shift를 두 번 누르면 무엇이든 검색 가능한 검색 창이 표시됩니다.
